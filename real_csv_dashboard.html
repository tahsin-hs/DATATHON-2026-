CSV-backed prototype

ViralCheck dashboard built from the raw trending files

This version loads the uploaded country CSVs directly in the browser, computes insights live, and redraws the charts from the selected dataset instead of relying on a precomputed JavaScript object.

This page is being opened as a local file. Browsers often block CSV loading from file://. Run a simple local server in the same folder, such as python -m http.server 8000, then open http://localhost:8000/real_csv_dashboard.html.
